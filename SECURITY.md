# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in the Pyrimid Protocol (contracts, SDK, API, or website), please report it responsibly.

**Email:** security@pyrimid.ai

**Do NOT** open a public GitHub issue for security vulnerabilities.

## Scope

- Smart contracts on Base (Registry, Catalog, Router, Treasury)
- `@pyrimid/sdk` npm package
- `pyrimid.ai` API endpoints
- MCP server endpoint
- Payment verification logic

## Response

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan within 7 days for critical issues.

## Bug Bounty

We do not currently operate a formal bug bounty program. Significant findings may be rewarded at our discretion.
